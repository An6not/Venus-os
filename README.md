# Venus-OS

Simple operating system simulation project.

---

## Описание

**Venus-OS** — это имитация операционной системы прямо в браузере. Рабочий стол, окна приложений и базовые интерактивные элементы реализованы на HTML, CSS и JavaScript.

Автор: **An6not**

---

## Запуск

Откройте (https://An6not.github.io/Venus-OS/)
в любом браузере с интернетом
